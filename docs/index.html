LocalPDF Studio

PDF를 PC 안에서 열고 정리하고 변환합니다. 계정, 광고, 클라우드 업로드 없이 민감한 문서 작업을 데스크톱에서 차분하게 처리하는 앱입니다.

GitHub에서 보기 개발 실행
100% 로컬 중심 PDF 내용은 사용자 장치 밖으로 나가지 않으며, 모든 편집/변환 작업을 로컬에서 처리합니다.
가볍고 빠른 데스크톱 Tauri 2, React 19, TypeScript, Rust로 ~15MB 설치 크기와 빠른 응답성을 유지합니다.
실무 PDF 워크플로 보기, 검색, 병합, 분할, OCR, 변환, 암호화, 워터마크까지 모두 내장되어 있습니다.

문서를 밖으로 보내지 않는 PDF 작업대

LocalPDF Studio는 로컬 파일 처리, 명확한 배포 경로, Windows 사용자 경험을 한 프로젝트 안에서 완성했습니다.

고성능 PDF 뷰어

PDF.js 기반 렌더링, 연속 스크롤, 페이지 맞춤, 확대/축소, 회전, 텍스트 선택/복사, 렌더 큐 최적화를 제공합니다.

대용량 PDF 스트리밍

250MB 이상 PDF는 커스텀 프로토콜과 Range 요청으로 청크 단위 스트리밍 로드하여 메모리와 속도를 최적화합니다.

다중 문서 탭 & 세션 복원

여러 PDF를 탭으로 동시 열기, 드래그 앤 드롭, 앱 재시작 시 열려 있던 문서와 뷰 상태를 자동으로 복원합니다.

PDF 병합 & 분할

qpdf 엔진으로 여러 PDF를 하나로 병합하거나 페이지 단위로 분할합니다. 드래그로 순서 변경 가능합니다.

페이지 추출 & 회전 저장

지정한 페이지 범위를 새 PDF로 추출하고, 페이지별 90°/180°/270° 회전을 영구 저장합니다.

암호화 & 복호화

256-bit AES로 PDF에 사용자/소유자 암호를 설정하거나, 암호를 입력하여 보호된 PDF의 잠금을 해제합니다.

OCR 텍스트 추출

Tesseract OCR 엔진으로 이미지 PDF에서 텍스트를 추출합니다. 언어 선택, DPI 설정을 지원합니다.

형식 변환

PDF→이미지(PNG/JPEG/WebP), PDF→TXT, 이미지→PDF 변환을 지원합니다. 워터마크/스탬프 적용도 가능합니다.

문서 비교 & 메타데이터

두 PDF의 텍스트를 추출하여 라인 단위 차이를 비교하고, PDF 메타데이터를 JSON으로 확인할 수 있습니다.

Windows 통합 배포

우클릭 메뉴 등록, NSIS/MSI/Portable ZIP 설치, Tauri updater 자동 업데이트를 지원합니다.

키보드 단축키 & 설정

Ctrl+O/F/B, Alt+←→, Ctrl+Tab/W 등 주요 기능을 단축키로 조작하고, 17가지 앱 설정을 UI로 제어합니다.

문서 탐색 도구

페이지 썸네일, 목차/아웃라인, 전문 검색, 최근 문서 목록으로 빠르게 원하는 위치를 찾을 수 있습니다.
