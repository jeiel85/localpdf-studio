LocalPDF Studio

PDF를 PC 안에서 열고 정리하고 변환합니다. 계정, 광고, 클라우드 업로드 없이 민감한 문서 작업을 데스크톱에서 차분하게 처리하는 앱입니다.

GitHub에서 보기 개발 실행
100% 로컬 중심 PDF 내용은 기본적으로 사용자 장치 밖으로 나가지 않습니다.
가볍고 빠른 데스크톱 Tauri, React, Rust로 Windows 앱의 응답성과 배포 크기를 관리합니다.
실무 PDF 워크플로 보기, 검색, 병합, 분할, OCR, 내보내기를 단계적으로 완성합니다.

문서를 밖으로 보내지 않는 PDF 작업대

LocalPDF Studio는 로컬 파일 처리, 명확한 배포 경로, Windows 사용자 경험을 한 프로젝트 안에서 함께 설계합니다.

PDF 보기와 탐색

PDF.js 기반 렌더링, 페이지 이동, 확대/축소, 회전, 사이드바 레이아웃을 제공합니다.

정리와 변환 준비

qpdf wrapper를 중심으로 병합, 분할, 추출, 암호화 작업 계층을 확장합니다.

OCR 확장 경로

Tesseract 탐지와 언어팩 확인 구조를 바탕으로 이미지 PDF 텍스트 추출을 준비합니다.

Windows 통합

우클릭 메뉴, NSIS, MSI, Portable ZIP까지 Windows 배포 흐름을 고려합니다.

업데이트 설계

Tauri updater와 GitHub Release 산출물을 연결하는 릴리즈 구조를 마련합니다.

개발자 친화 문서

기능 명세, 아키텍처, 보안, 릴리즈 문서를 함께 관리해 다음 작업을 빠르게 이어갑니다.

제품화까지 이어지는 로드맵

지금은 초기 골격 단계이며, 검증 가능한 단위로 PDF 뷰어 완성도와 작업 엔진을 넓혀갑니다.

P0
저장소 안정화 의존성 설치, 타입 검사, 앱 빌드, 기본 PDF 렌더링을 검증합니다.
P1
뷰어 경험 완성 썸네일, 목차, 검색, 최근 문서, 단축키를 추가합니다.
P2
PDF 작업 엔진 병합, 분할, 추출, 회전 저장, 암호화, 작업 큐를 구현합니다.
P3/P4
OCR과 배포 OCR, 변환, 설치 파일, 업데이트 서명, 릴리즈 자동화를 마무리합니다.

로컬 PDF 도구를 직접 빌드하세요

저장소를 클론한 뒤 Node.js LTS, Rust stable, WebView2 환경에서 바로 개발을 시작할 수 있습니다.
