LocalPDF Studio

광고도, 계정도, 클라우드 업로드도 없습니다. PDF 작업은 사용자의 PC 안에서만 일어납니다. 뷰어 · 병합 · 분할 · OCR · 폼 채우기 · 메타데이터 편집 · 분할 비교까지 한 앱에 모두 담은 데스크톱 PDF 스튜디오입니다.

Windows 10+ macOS 10.15+ (M1+Intel) Linux (AppImage · deb · rpm)
최신 버전 다운로드 GitHub 저장소
100% 로컬 처리 PDF 내용은 사용자 장치 밖으로 나가지 않습니다. 모든 편집·변환이 PC 안에서 실행됩니다.
가볍고 빠름 Tauri 2 + React 19 + Rust. ~15MB 설치 크기, 메인 JS 117KB. 즉시 응답하는 데스크톱 경험.
완성된 PDF 워크플로 뷰어부터 OCR · 폼 채우기 · 분할 비교 · 워터마크까지 한 앱에 통합되어 있습니다.

문서를 밖으로 보내지 않는 PDF 작업대

뷰어 · 편집 · OCR · 변환 · 비교 · 폼 · 책갈피. 전형적인 데스크톱 PDF 도구에 기대하는 거의 모든 기능이 v0.10.0에 들어 있습니다.

📖 고성능 PDF 뷰어

PDF.js 기반 렌더링, 연속 스크롤, 페이지 맞춤 3종, 확대/축소, 회전, 텍스트 선택/복사. 렌더 큐로 메인 스레드 보호.

🌊 대용량 PDF 스트리밍

250MB+ PDF는 커스텀 프로토콜 + HTTP Range로 청크 단위 로드. base64 인플레이션 회피, 메모리 최적화.

📑 다중 문서 탭 + 세션 복원

여러 PDF를 탭으로 동시 열고, 재시작 시 페이지·배율·회전·레이아웃까지 복원합니다.

🔀 병합 · 분할 · 추출 · 회전

qpdf 엔진. 드래그로 순서 변경, 페이지 범위 지정, 256-bit AES 암호화, 압축, 페이지별 인라인 회전.

🔍 OCR + 검색 가능 PDF

Tesseract 자동 설치 (SHA-256 검증). 스캔 PDF를 텍스트 레이어 합성된 검색 가능 PDF로 변환합니다.

✏️ 페이지 편집기

드래그로 재정렬, 삭제, 다른 PDF에서 페이지 삽입, 인라인 회전. 약 1.2초마다 편집 상태 자동 백업.

🏷️ 메타데이터 · 폼 · 책갈피

Title/Author/Subject/Keywords 편집. AcroForm 필드 채우기. PDF별 로컬 책갈피 (원본 미수정).

🔀 분할 뷰 비교

두 PDF를 좌/우 동시 표시. 페이지별 텍스트 차이를 분석해 다른 페이지로 빠르게 이동합니다.

🛡️ 개인정보 보안 마스킹 (블랙아웃)

100% 오프라인 로컬 처리. 안전한 래스터화(300DPI 페이지 재구축) 기술을 적용하여 하단의 텍스트와 벡터 레이어를 영구적으로 파괴하고 정보 유출을 완벽하게 방어합니다.

✍️ 표준 PDF 하이라이트 주석

여러 페이지를 넘나드는 자연스러운 드래그 선택과 저수준 pdf-lib 엔진 임베딩으로 Adobe Acrobat, Chrome, macOS Preview 등 기성 뷰어의 주석 패널과 완벽한 양방향 호환을 보장합니다.

🎨 PDF 워터마크 삽입

다른 PDF 파일을 Overlay/Underlay 방식으로 레이어 합성하여 신속하게 문서 전체에 고품질 워터마크를 적용합니다.

🌐 다국어 (한·영·일)

브라우저 언어 자동 감지. 설정에서 즉시 전환. 한국어 / 영어 / 일본어 UI.

🔒 보안 강화

CSP 활성화, 경로 검증 일원화, 자동 설치 SHA-256 검증, JSON atomic write, PDF.js JS 차단.

⌨️ 키보드 단축키 25+

Ctrl+O/F/B/G/L/P, +/-/0, Home/End, Alt+←→, F1 도움말. 설정 패널 20+ 옵션.

다운로드

OS에 맞는 산출물을 선택하세요. 모든 빌드는 GitHub Release에서 직접 받을 수 있고, Windows는 자동 업데이트가 활성화되어 있습니다.

Windows 10+
NSIS .exe MSI Portable .zip

NSIS 인스톨러는 자동 업데이트와 우클릭 메뉴를 지원합니다.

Windows 다운로드
macOS 10.15+ 무서명
Universal DMG .app

Apple Silicon + Intel 단일 DMG. 첫 실행 시 Gatekeeper 우회 필요 (아래 안내).

macOS 다운로드
Linux
AppImage .deb .rpm

대부분 배포판 호환. AppImage는 다운로드 후 chmod +x로 즉시 실행.

Linux 다운로드
⚠️ macOS Gatekeeper 우회 안내
macOS 빌드는 현재 Apple Developer 서명/공증을 받지 않았습니다 ($99/년 비용 부담). 첫 실행 시 “확인되지 않은 개발자” 경고가 표시됩니다. 다음 두 가지 방법 중 하나로 우회하세요:
① Finder → Applications → 앱 우클릭 → “열기” → 대화상자에서 다시 “열기”
② 터미널에서 xattr -dr com.apple.quarantine "/Applications/LocalPDF Studio.app"
자세한 절차는 INSTALL.md 참고.

패키지 매니저로 설치

각 매니페스트가 작성되어 있어 등록되는 대로 다음 명령으로 설치/업데이트할 수 있습니다. 상세 절차는 packaging/ 폴더의 README를 참고하세요.

winget
winget install jeiel85.LocalPDFStudio ● 매니페스트 작성 완료, microsoft/winget-pkgs PR 대기
Chocolatey
choco install localpdf-studio ● 패키지 작성 완료, community 큐 제출 예정
Homebrew (tap)
brew tap jeiel85/tap brew install --cask localpdf-studio ● Cask 작성 완료, personal tap 우선
Snap Store
sudo snap install localpdf-studio ● snapcraft.yaml 준비, 등록 대기
AUR (Arch / Manjaro)
yay -S localpdf-studio-bin ● PKGBUILD 준비, AUR 등록 대기
GitHub Release
gh release download --repo jeiel85/localpdf-studio ● Live · 3-OS 매트릭스 자동 빌드
