LocalPDF Studio

A desktop PDF studio that runs entirely on your machine. No accounts, no ads, no cloud uploads — view, search, merge, split, OCR, and convert PDFs with the privacy guarantees of local computing. Available for Windows, macOS, and Linux.

Download latest View on GitHub

Built for serious PDF work

LocalPDF Studio combines a high-performance PDF viewer with a complete local PDF toolkit. Every operation happens on your device.

Fast PDF viewer

PDF.js rendering with continuous scroll, fit modes, zoom/rotate, text selection, and a render queue tuned to avoid scroll jank.

Large PDF streaming

Documents over 250 MB stream via a custom protocol with HTTP Range requests, avoiding base64 inflation and keeping memory low.

Multi-document tabs & session restore

Open many PDFs side by side; on restart, your tabs, page, scale, rotation, and layout are restored.

Merge / split / extract / rotate

qpdf-powered operations. Drag-reorder, page-range syntax, 256-bit AES encryption, linearize + object stream compression.

OCR via Tesseract

One-click auto-install with SHA-256 integrity verification. Generate searchable PDFs by combining PDF.js rendering and Tesseract's PDF output mode.

Page editor

Drag to reorder, delete, insert pages from other PDFs, inline rotate per page. Auto-saves edit state every ~1 second to recover from crashes.

Metadata, forms, bookmarks

Edit Title/Author/Subject/Keywords; fill AcroForm fields; manage per-file bookmarks stored locally.

Side-by-side compare

Open two PDFs in a split view. Per-page text diff analysis highlights pages that differ for quick navigation.

Watermark & highlight

Overlay or stamp another PDF as a watermark. Add page-level color bands as quick visual highlights.
