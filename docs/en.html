LocalPDF Studio

A desktop PDF studio that runs entirely on your machine. No accounts, no ads, no cloud uploads — view, search, merge, split, OCR, edit metadata, fill forms, and compare side-by-side. All in one app for Windows, macOS, and Linux.

Windows 10+ macOS 10.15+ (M1+Intel) Linux (AppImage · deb · rpm)

100% local processing PDF content never leaves your device. All edits and conversions happen on your PC.
Light & fast Tauri 2 + React 19 + Rust. ~15MB install, 117KB main JS bundle. Instant desktop responsiveness.
Complete PDF workflow Viewer · OCR · forms · compare · watermark · metadata — all in one integrated app.

Built for serious PDF work

Viewer · editor · OCR · conversion · compare · forms · bookmarks · Fill & Sign. Almost everything you'd expect from a desktop PDF tool is in v0.17.0 — now with free-form stamps and handwritten or image signatures.

📖 Fast PDF viewer

PDF.js rendering with continuous scroll, three fit modes, zoom/rotate, text selection. Render queue protects the main thread.

🌊 Large PDF streaming

Documents over 250 MB load via a custom protocol with HTTP Range requests. No base64 inflation, low memory.

📑 Multi-document tabs + session restore

Open many PDFs as tabs; on restart, page, scale, rotation, and layout are restored automatically.

🔀 Merge · split · extract · rotate

qpdf-powered. Drag-reorder, page-range syntax, 256-bit AES encryption, compression, per-page inline rotation.

🔍 OCR + searchable PDF

Tesseract auto-install with SHA-256 verification. Convert scanned PDFs into searchable PDFs with a text layer.

✏️ Page editor

Drag to reorder, delete, insert pages from other PDFs, inline rotate. Auto-saves edit state every ~1.2s.

🏷️ Metadata · forms · bookmarks

Edit Title/Author/Subject/Keywords. Fill AcroForm fields. Per-file bookmarks stored locally (original PDF untouched).

🔀 Side-by-side compare

Open two PDFs in a split view. Per-page text diff analysis lets you jump straight to pages that differ.

🛡️ Secure Redaction (Blackout)

100% offline local processing. Secure Rasterization (300DPI page-rebuilding strategy) permanently destroys underlying text/vector layers to prevent metadata leak.

✍️ Standard Highlight Annotation

Seamless multi-page text dragging and low-level pdf-lib embedding for perfect bidirectional compatibility with Adobe Acrobat, Chrome, and macOS Preview.

🎨 PDF Watermarking

Composite other PDF files as overlays or underlays to quickly apply high-quality watermarks across the document.

🌐 Multilingual (KO/EN/JA)

Auto-detects browser language. Switch instantly in Settings. Korean / English / Japanese UI.

🔒 Security hardened

CSP enabled, unified path validation, SHA-256 verified installer downloads, atomic JSON writes, PDF.js JS disabled.

⌨️ 25+ keyboard shortcuts

Ctrl+O/F/B/G/L/P, +/-/0, Home/End, Alt+←→, F1 for help. 20+ settings options.

⚠️ macOS Gatekeeper bypass
macOS builds are currently unsigned because Apple Developer notarization costs $99/year. On first launch you'll see “cannot be opened because the developer cannot be verified.” Bypass with either:
① Finder → Applications → right-click the app → "Open" → confirm in the dialog
② Terminal: xattr -dr com.apple.quarantine "/Applications/LocalPDF Studio.app"
