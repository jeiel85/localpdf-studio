# Product Brief

LocalPDF Studio는 개인 사용과 공개 포트폴리오를 모두 고려한 Windows 우선 PDF 데스크톱 앱입니다.

## 목표

- 광고, 계정, 클라우드 업로드 없는 로컬 우선 PDF 처리
- 상용화 가능한 UI/UX와 배포 구조
- Windows 우클릭 메뉴와 연동되는 생산성 도구
- PDF 보기, 병합, 분할, 압축, OCR, 변환, 암호화, 메타데이터 편집까지 확장 가능한 구조

## 비목표

- 광고 SDK
- 사용자 추적 분석
- 서버 업로드 기반 PDF 처리
- DRM 우회 또는 접근 제한 우회
