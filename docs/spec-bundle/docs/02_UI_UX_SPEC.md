## 2. 화면 목록

### 2.1 Home

목적: PDF를 열거나 빠른 작업을 시작한다.

구성:

- 최근 문서
- 빠른 작업 카드
  - PDF 병합
  - PDF 분할
  - PDF 압축
  - PDF OCR
  - 이미지에서 PDF 만들기
  - PDF를 이미지로 변환
- 드래그 앤 드롭 영역
- 업데이트 알림
- 최근 작업 결과

### 2.2 Reader

목적: PDF 열람과 검색.

구성:

- 상단 툴바
- 좌측 패널
  - 썸네일
  - 목차
  - 검색
  - 주석
- 중앙 페이지 뷰
- 우측 인스펙터
  - 문서 정보
  - 선택 객체 정보
  - 주석 속성
- 하단 상태바

### 2.3 Page Organizer

목적: 페이지 재정렬, 삭제, 회전, 추출, 삽입.

구성:

- 페이지 그리드
- 다중 선택
- 드래그 앤 드롭
- 페이지 범위 입력
- 작업 미리보기
- 적용 전 변경 요약
- 저장/다른 이름으로 저장

### 2.4 Merge Wizard

단계:

1. 파일 추가
2. 순서 변경
3. 북마크/목차 병합 옵션
4. 출력 파일 선택
5. 실행 및 결과

### 2.5 Split Wizard

방식:

- 페이지 범위별 분할
- N페이지마다 분할
- 선택 페이지 추출
- 북마크 기준 분할
- 홀수/짝수 분리

### 2.6 OCR Wizard

단계:

1. 입력 파일 선택
2. 페이지 범위 선택
3. 언어 선택
4. 출력 방식 선택
   - 검색 가능한 PDF
   - TXT
   - PDF + TXT
5. 실행
6. OCR 결과 확인

### 2.7 Batch Jobs

구성:

- 작업 큐
- 파일별 진행률
- 전체 진행률
- 실패 사유
- 재시도
- 결과 폴더 열기
- 리포트 저장

### 2.8 Settings

섹션:

- General
- Viewer
- Context Menu
- OCR
- Updates
- Privacy
- Advanced
- About

## 3. 주요 UX 규칙

- 모든 파괴적 작업은 기본적으로 사본 저장을 유도한다.
- 원본 덮어쓰기 시 명확한 경고와 백업 옵션을 제공한다.
- 긴 작업은 진행률, 현재 파일, 현재 페이지, 취소 버튼을 표시한다.
- 실패 메시지는 사용자가 할 수 있는 다음 행동을 포함한다.
- `Ctrl+S`는 수정된 문서를 저장하고, 최초 저장은 `Save As`를 연다.
- `Ctrl+Shift+S`는 항상 `Save As`를 연다.
- `Ctrl+O`는 파일 열기.
- `Ctrl+F`는 검색.
- `Ctrl+P`는 인쇄.
- `Ctrl+Plus/Minus`는 확대/축소.
- `F11`은 전체 화면.
- `Esc`는 전체 화면/모달 닫기.

## 4. 접근성

- 키보드만으로 주요 기능 접근 가능
- 명도 대비 AA 수준 지향
- 화면 읽기 프로그램용 버튼 라벨
- 모든 아이콘은 텍스트 툴팁 제공
- 작은 화면에서도 최소 사용 가능
- 다크 모드와 라이트 모드 지원

## 5. 상태 표시

상태바 정보:

- 파일명
- 현재 페이지/전체 페이지
- 확대율
- 선택 모드
- 저장 상태
- 작업 큐 상태
- 업데이트 상태

## 6. 오류 메시지 예시

| 상황 | 메시지 |
|---|---|
| 암호화 PDF | 이 PDF는 암호가 필요합니다. 파일을 열려면 문서 암호를 입력하세요. |
| 파일 손상 | PDF 구조가 손상되어 일부 페이지를 읽지 못했습니다. 복구 모드로 다시 열 수 있습니다. |
| 저장 실패 | 파일을 저장하지 못했습니다. 파일이 다른 프로그램에서 열려 있거나 권한이 없을 수 있습니다. |
| OCR 언어 없음 | 선택한 OCR 언어 데이터가 설치되어 있지 않습니다. 설정에서 언어 데이터를 추가하세요. |
| 업데이트 실패 | 업데이트를 설치하지 못했습니다. 네트워크 또는 서명 검증 오류를 확인하세요. |
