# 00. Product Brief

## 제품명

**LocalPDF Studio**

## 한 줄 설명

로컬에서 안전하게 PDF를 열고, 정리하고, 변환하고, OCR하고, Windows 우클릭 메뉴에서 즉시 작업할 수 있는 광고 없는 데스크톱 PDF 도구.

## 사용자

1. 개인 사용자
   - 문서 병합, 분할, 회전, 압축, OCR, 이미지 변환을 자주 수행한다.
   - 온라인 PDF 사이트에 민감 문서를 업로드하고 싶지 않다.
2. 개발자/1인 개발자
   - GitHub에 공개 가능한 수준의 데스크톱 앱 포트폴리오가 필요하다.
   - 바이브 코딩으로 유지보수 가능한 구조를 원한다.
3. 소규모 사무 사용자
   - 설치 파일, 우클릭 메뉴, 자동 업데이트, 기본적인 안정성이 필요하다.

## 제품 원칙

- Local-first: 기본 PDF 작업은 모두 로컬에서 수행한다.
- No Ads: 광고 SDK와 추적 SDK를 넣지 않는다.
- No Account: 계정 없이 사용 가능해야 한다.
- Safe by Default: 원본 파일 덮어쓰기는 기본 금지한다.
- Commercial-grade: 설치, 업데이트, 로그, 테스트, 문서화, 예외 처리까지 포함한다.
- License-conscious: 상용화 가능성을 해치기 쉬운 라이선스는 기본 번들에서 제외한다.

## 1차 플랫폼

- Windows 10/11 x64
- 추후 macOS, Linux 대응 가능 구조

## 성공 기준

- 300MB 이상의 PDF를 열어도 UI가 멈추지 않는다.
- 1,000페이지급 PDF의 썸네일 목록을 가상화 렌더링으로 다룬다.
- PDF 병합/분할/회전/추출/압축/OCR 작업이 취소 가능하다.
- Windows Explorer 우클릭 메뉴에서 핵심 작업을 실행할 수 있다.
- 설치 파일과 Portable ZIP을 GitHub Release에 자동 업로드한다.
- 앱 자체 업데이트는 서명 검증을 통과한 릴리즈만 설치한다.
