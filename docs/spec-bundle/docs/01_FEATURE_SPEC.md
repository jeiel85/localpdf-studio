# 01. Feature Specification

## 1. PDF 보기

### 필수

- PDF 열기
- 최근 문서
- 탭 기반 다중 문서
- 단일 페이지 보기
- 연속 스크롤 보기
- 두 페이지 보기
- 맞춤 확대
  - 페이지 맞춤
  - 폭 맞춤
  - 실제 크기
  - 사용자 지정 배율
- 확대/축소
- 페이지 이동
- 페이지 번호 입력 이동
- 북마크/목차 표시
- 썸네일 사이드바
- 텍스트 검색
- 검색 결과 목록
- 선택 텍스트 복사
- 페이지 회전 보기
- 전체 화면
- 다크 모드
- 인쇄

### 고급

- 읽기 모드
- 프레젠테이션 모드
- 마지막 위치 기억
- 문서별 보기 설정 기억
- 대용량 PDF lazy loading
- 손상 PDF 열기 시 복구 시도

## 2. 페이지 정리

- 페이지 썸네일 드래그 재정렬
- 선택 페이지 삭제
- 선택 페이지 회전
- 선택 페이지 추출
- 선택 페이지 복제
- 빈 페이지 삽입
- 외부 PDF 페이지 삽입
- 여러 PDF 병합
- 페이지 범위 기반 분할
- N페이지마다 분할
- 북마크 기준 분할
- 홀수/짝수 페이지 추출
- 역순 재정렬

## 3. 변환

### PDF에서 내보내기

- PDF → PNG
- PDF → JPG
- PDF → TXT
- PDF → Markdown, 후순위
- PDF → 이미지 ZIP

### PDF로 만들기

- 이미지 → PDF
- 여러 이미지 → 단일 PDF
- 텍스트 파일 → PDF, 후순위

## 4. OCR

- 스캔 PDF OCR
- 이미지 PDF OCR
- 선택 페이지 OCR
- 언어 선택
  - Korean
  - English
  - Japanese, optional
  - Chinese, optional
- OCR 결과 검색 가능 PDF 생성
- OCR 텍스트 추출
- OCR 작업 큐
- OCR 진행률 표시
- OCR 취소
- OCR 결과 미리보기

## 5. 주석/마크업

- 하이라이트
- 밑줄
- 취소선
- 자유형 펜
- 사각형
- 원
- 화살표
- 텍스트 박스
- 스티키 노트
- 스탬프
- 서명 이미지 삽입
- 주석 목록
- 주석 필터
- 주석 포함 저장
- 주석만 내보내기, 후순위

## 6. 편집/보안

- 메타데이터 보기/수정
- 문서 제목, 작성자, 주제, 키워드 수정
- 파일 암호화
- 열기 암호 설정
- 권한 설정, 후순위
- 암호 제거
  - 사용자가 암호를 알고 입력한 경우만 허용
  - 우회/크랙 기능 금지
- 민감정보 마스킹
  - 시각적 덮기와 실제 내용 제거를 구분하여 표시
  - 실제 redaction은 별도 검증 필요
- 워터마크 추가
- 페이지 번호 추가
- 머리글/바닥글 추가

## 7. 압축/최적화

### 기본 압축

- qpdf 기반 구조 최적화
- linearization
- object stream 최적화
- 불필요 메타데이터 제거 옵션
- 첨부파일 제거 옵션

### 고급 압축

- 이미지 재샘플링
- 이미지 품질 조절
- 흑백 변환
- 이 기능은 라이선스 안전한 구현을 별도 검토 후 추가한다.

## 8. 비교

- 두 PDF 시각 비교
- 페이지별 차이 표시
- 텍스트 차이 비교
- 메타데이터 차이 비교
- 변경 페이지 목록

## 9. 배치 작업

- 여러 PDF 병합
- 여러 PDF 각각 압축
- 여러 PDF 각각 OCR
- 여러 PDF를 이미지로 변환
- 파일명 규칙 적용
- 출력 폴더 지정
- 실패 파일 목록 제공
- 작업 결과 리포트 저장

## 10. 설정

- 언어
- 테마
- 기본 저장 위치
- 최근 파일 보관 개수
- 자동 업데이트 설정
- 우클릭 메뉴 활성화/비활성화
- OCR 언어 데이터 설치 상태
- 임시 파일 자동 삭제 정책
- 로그 레벨
- 실험 기능 on/off

## 11. 비기능 요구사항

- 원본 덮어쓰기 기본 금지
- 모든 장시간 작업 취소 가능
- 작업 실패 시 원본 파일 보존
- 임시 파일은 앱 전용 temp 디렉터리에 저장
- crash 후 임시 파일 정리 안내
- 민감 정보는 로그에 기록하지 않음
- 오프라인 사용 가능
