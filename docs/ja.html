LocalPDF Studio

アカウントも広告もクラウドアップロードもありません。 PDF の処理はすべてユーザーの PC 内で完結します。 ビューア · 結合 · 分割 · OCR · フォーム入力 · メタデータ編集 · 左右比較まで、 1 つのアプリにまとめたデスクトップ PDF スタジオです。

Windows 10+ macOS 10.15+ (M1+Intel) Linux (AppImage · deb · rpm)
最新版をダウンロード GitHub で見る
100% ローカル処理 PDF の内容はデバイスの外に出ません。すべての編集と変換が PC 内で実行されます。
軽量で高速 Tauri 2 + React 19 + Rust。インストールサイズ ~15MB、メイン JS バンドル 117KB。即応するデスクトップ体験。
完成された PDF ワークフロー ビューアから OCR、フォーム、比較、透かし、メタデータまで、1 つのアプリに統合。

本格的な PDF 作業のために

ビューア · 編集 · OCR · 変換 · 比較 · フォーム · しおり · Fill & Sign。 デスクトップ PDF ツールに期待されるほぼすべての機能が v0.17.0 に含まれています。自由スタンプと手書き／画像署名にも対応しました。

📖 高速 PDF ビューア

PDF.js による描画。連続スクロール、3 種のフィットモード、ズーム / 回転、テキスト選択。レンダーキューでメインスレッドを保護。

🌊 大容量 PDF のストリーミング

250MB 超の文書はカスタムプロトコル + HTTP Range で逐次読み込み。base64 膨張を回避しメモリを抑えます。

📑 マルチタブ · セッション復元

複数の PDF を同時に開け、再起動時にはタブ・ページ・倍率・回転・レイアウトを自動復元。

🔀 結合 · 分割 · 抽出 · 回転

qpdf ベース。ドラッグ並び替え、ページ範囲記法、256-bit AES 暗号化、圧縮、ページごとの回転。

🔍 OCR + 検索可能 PDF

SHA-256 検証付きの Tesseract 自動インストール。スキャン PDF をテキストレイヤー付きの検索可能 PDF に変換。

✏️ ページエディタ

ドラッグで並び替え、削除、別 PDF からの挿入、ページ単位の回転。約 1.2 秒ごとに編集状態を自動保存。

🏷️ メタデータ · フォーム · しおり

タイトル / 著者 / 件名 / キーワードを編集。AcroForm フィールドの入力。ファイルごとのしおりをローカル保存。

🔀 左右並べて比較

2 つの PDF を分割表示。ページごとのテキスト差分解析で差異のあるページに素早く移動。

🛡️ 個人情報安全マスキング（ブラックアウト）

100%オフラインローカル処理。安全なラスタライズ（300DPI再構築戦略）により、下層のテキストやベクターレイヤーを永久的に破壊し、個人情報の漏洩を根本的に防御。

✍️ 標準PDFハイライト注釈

複数のページをまたぐ自然なドラッグ選択と、低レベルのpdf-lib埋め込みにより、Adobe Acrobat、Chrome、macOS Previewなどの外部ビューアの注釈パネルと完璧な双方向互換性を確保。

🎨 PDF透かし合成

別のPDFファイルを前面または背面にレイヤー合成し、ドキュメント全体に高品質な透かしを迅速に適用します。

🌐 多言語 (韓・英・日)

ブラウザの言語を自動検出。設定から即時切り替え。韓国語 / 英語 / 日本語 UI。

🔒 セキュリティ強化

CSP 有効化、パス検証の一元化、自動インストーラの SHA-256 検証、JSON アトミック書き込み、PDF.js の JS 無効化。

⌨️ キーボードショートカット 25+

Ctrl+O/F/B/G/L/P、+/-/0、Home/End、Alt+←→、F1 ヘルプ。20+ 個の設定オプション。

⚠️ macOS Gatekeeper の回避手順
macOS ビルドは Apple Developer の年額 $99 のコスト負担により、現状は未署名です。 初回起動時に「開発元を確認できないため開けません」と表示されます。次のいずれかで回避してください:
① Finder → アプリケーション → アプリを右クリック → 「開く」 → ダイアログで「開く」を選択
② ターミナル: xattr -dr com.apple.quarantine "/Applications/LocalPDF Studio.app"
詳しい手順は INSTALL.md を参照。
