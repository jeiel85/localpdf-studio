LocalPDF Studio

ユーザーの PC 内だけで PDF を扱うデスクトップアプリ。 アカウント・広告・クラウドアップロードは一切ありません。 閲覧、検索、結合、分割、OCR、変換、暗号化までローカルで完結します。 Windows / macOS / Linux に対応。

最新版をダウンロード GitHub で見る

本格的な PDF 作業のために

LocalPDF Studio は高性能 PDF ビューアと包括的なローカル PDF ツールを統合しました。 すべての操作はお使いのデバイス上で実行されます。

高速 PDF ビューア

PDF.js による描画。連続スクロール、フィットモード、ズーム / 回転、テキスト選択、スクロールジャンクを避けるレンダーキュー。

大容量 PDF のストリーミング

250MB 超の文書はカスタムプロトコル + HTTP Range で逐次読み込み。base64 膨張を回避しメモリを抑えます。

マルチタブ・セッション復元

複数の PDF を同時に開け、再起動時にはタブ・ページ・倍率・回転・レイアウトを自動復元。

結合 / 分割 / 抽出 / 回転

qpdf ベース。ドラッグ並び替え、ページ範囲記法、256-bit AES 暗号化、linearize + object stream 圧縮。

Tesseract OCR

SHA-256 検証付きのワンクリック自動インストール。PDF.js レンダリング + Tesseract の PDF 出力で検索可能 PDF を生成。

ページエディタ

ドラッグで並び替え、削除、別 PDF からの挿入、ページ単位の回転。約 1 秒ごとに編集状態を自動保存。

メタデータ・フォーム・しおり

タイトル / 著者 / 件名 / キーワードを編集。AcroForm フィールドの入力。ファイルごとのしおりをローカル保存。

左右並べて比較

2 つの PDF を分割表示。ページごとのテキスト差分解析で差異のあるページに素早く移動。

透かし・ハイライト

別 PDF を透かし / スタンプとして重ね合わせ。ページ単位の色バンドを簡易ハイライトとして追加。

ダウンロード

お使いのプラットフォーム向けのファイルをお選びください。 GitHub の最新リリース から直接ダウンロードできます。

プラットフォーム形式インストール
Windows 10+ NSIS .exe / MSI / Portable ZIP インストーラを実行するか winget install jeiel85.LocalPDFStudio
macOS 10.15+未署名 Universal DMG (Apple Silicon + Intel) DMG をマウントし Applications にドラッグ。初回起動は右クリック→開く
Linux AppImage / .deb / .rpm sudo apt install ./*.deb · chmod +x *.AppImage

macOS ビルドは Apple Developer の年額 $99 のコストが発生するため、現状は未署名です。 Gatekeeper 回避手順は インストールガイド をご覧ください。
