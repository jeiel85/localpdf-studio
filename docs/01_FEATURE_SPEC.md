- PDF 열기
- 페이지 이동
- 확대/축소
- 회전 표시
- 썸네일
- 목차
- 검색
- 탭
- 최근 문서

- 병합
- 분할
- 추출
- 회전 저장
- 압축
- 암호 설정/해제
- 메타데이터 편집

- 스탬프/서명 이미지

- Image to PDF
